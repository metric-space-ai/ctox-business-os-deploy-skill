# CTOX Business OS Deploy Skill

GitHub-hosted Codex skill for installing, connecting, and verifying CTOX
Business OS remote-control deployments.

Install into Codex:

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo metric-space-ai/ctox-business-os-deploy-skill \
  --path ctox-business-os-deploy
```

Restart Codex after installation.

The installable skill lives in `ctox-business-os-deploy/`.
